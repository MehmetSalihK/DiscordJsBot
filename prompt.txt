Build a complete Discord bot dashboard from scratch, accessible at
http://localhost:5173/dashboard
.
The dashboard must be fully redesigned (from zero) with a modern UI/UX and real-time configuration panels for all bot features.
Everything should be in French, responsive, clean, and smooth — with subtle animations, tooltips, shadows, rounded cards, and a dark theme aesthetic.

🌙 🖥️ Design Specifications (Dark UI / UX / UIX)

General theme:

Modern, flat dark interface

Use shades of dark gray (#0f0f10 / #1c1c1f) as background

Use accent color: #5865F2 (Discord blue)

Smooth gradient accents and glowing button hovers

Fonts: Inter / Poppins / Nunito Sans

Rounded corners (border-radius: 1.2rem)

Drop shadows for depth

Animated transitions (Framer Motion or GSAP)

Full responsiveness (desktop, tablet, mobile)

⚡ Technical Requirements

Frontend: React.js + Vite (npm create vite@latest)

Styling: Tailwind CSS + Framer Motion (for animations)

Backend: Express.js or Next.js API routes

Database: MongoDB (mongoose) + JSON backups in ./json/ & ./data/

Auth: Discord OAuth2 (discord.js, passport-discord)

Realtime updates: Socket.io (for XP, logs, config sync)

🧠 Animations et Effets

🗝️ Login System

First page = /login (OAuth2 Discord)

Shows “Connecte-toi avec Discord pour accéder à ton tableau de bord”

After login → redirects to /dashboard

Store Discord token in secure session (cookie)

Display user info (avatar + pseudo) in navbar
